**Автор:** Иван Иванов (пример)

## Описание
Desktop-приложение для конвертации валют с использованием актуальных курсов из внешнего API (ExchangeRate API). Поддерживает сохранение истории конвертаций в JSON и красивый графический интерфейс.

## Требования
- Python 3.7+
- Библиотеки: `requests`, `tkinter` (встроен в Python)
